# Finnhub

> Free realtime API for stocks, forex, and cryptocurrency: real-time quotes and candles, company fundamentals, estimates, ownership, filings, transcripts, ETFs/mutual funds/bonds/indices, economic data, and alternative datasets. REST base URL https://finnhub.io/api/v1 plus a real-time WebSocket at wss://ws.finnhub.io. All requests are authenticated with an API key passed as the `token` query parameter or the `X-Finnhub-Token` header (free key via https://finnhub.io/register). A hard cap of 30 API calls/second applies on top of per-plan limits; HTTP 429 signals a breached limit.

Generated by API Evangelist from the Finnhub catalog entry (method: generated — Finnhub publishes no llms.txt; /llms.txt returns the HTML site shell).
